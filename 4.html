1. 원시 웹

인터넷이 도시라면, 웹은 도시 위에 있는 건물 하나.
즉, 인터넷이 운영체제라면 웹은 운영체제 위에서 동작하는 하나의 앱.
웹과 동급인 기술은 상당히 많이 있다. 예를 들면 FTP, email 등.

1960 인터넷 탄생
1990 웹의 시작
팀 버너스 리의 원시 웹
웹의 메소포타미아


2. 서버와 클라이언트

인터넷이 동작하기 위해서는 최소 2대의 컴퓨터가 있어야 한다.
a컴퓨터에는 웹브라우저, b컴퓨터에는 웹 서버 프로그램이 깔려있다.

웹 서버는 웹브라우저와 마찬가지로 인터넷으로 다운받아 설치한 후, 실행하는 프로그램의 일종.
웹 서버를 배우는 과정 : 웹 호스팅으로 시작, 웹 서버 설치


웹 서버(b)와 웹 브라우저(a)의 통신(윈도우)은 어떻게 일어나는가?
웹 서버가 설치된 b컴퓨터 특정 디스크에 index.html 파일이 있음. a컴퓨터가 웹 서버에 접속하여 html파일을 불러 오려면?

ip주소/index.html이라는 주소를 입력하고 엔터를 친다.

컴b(서버)의 ip주소 필요.
그렇다면 컴a(클라이언트)는 어떻게 ip주소를 알아내는가?
이렇게.

*모든 컴퓨터는 자기 자신을 127.0.0.1로 ip를 잡는다고 약속함.
